# Protected files — read-only for the agent.
#
# Entries are ABSOLUTE paths. The entrypoint enforces root ownership
# and read-only modes after the chown sweep that hands the rest of
# /data/platform and /data/shell/ to the mobius user.
#
# Two categories live here:
#
# 1. **Credential surfaces** — files that handle password / token
#    input. Agent could exfiltrate by tampering. Keep root-owned.
#
# 2. **Boot / restore infra** — the platform container always starts
#    from baked entrypoint.sh, and recovery_restore.sh is the baked
#    restore tool. recoveryd is the separate recovery floor for HTTP.
