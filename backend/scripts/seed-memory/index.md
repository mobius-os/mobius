# Memory

This is your **Home** map — the root of your knowledge graph at
`/data/shared/memory/`. It is injected at the start of each session. Everything
below is reachable from here; follow a wiki-link by `Read`-ing
`notes/<slug>.md` or `mocs/<slug>.md` when you need the detail.

You record what is **useful for the future** — durable facts about the user,
hard-won bugs, and platform contracts — not everything. See
[[how-the-memory-graph-works]] for the rules.

## Maps

- [[about-the-user]] — who the user is: preferences, interests, personality,
  how they want you to work. *Grows through use — start here when a chat hints
  at a durable preference.*
- [[building-mobius-apps]] — gotchas + contracts for building mini-apps that
  took real effort to learn.
- [[mobius-platform]] — how the Möbius platform itself behaves (cron, shell,
  backend, storage, git).
- [[maintaining-memory]] — how this graph works and how to grow it.

## Inbox

Raw same-day observations live in `inbox.md` (also injected). The nightly
dreaming pass folds them into notes here, then clears the inbox.
